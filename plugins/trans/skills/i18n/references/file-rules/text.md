# 纯文本和配置文件翻译规则

## 纯文本（.txt）

- 全文翻译
- 保留空行、段落分隔、列表缩进等格式
- 术语保护生效
- 代码块（如有）按 Markdown 代码块规则处理

## 配置文件（.conf / .ini / .cfg）

### INI 格式示例

```ini
; 注释需要翻译
[database]           ; 节名不翻译
host = localhost     ; 键名不翻译，值翻译
port = 5432          ; 数字不翻译
name = 生产数据库    ; 字符串值翻译
debug = true         ; 布尔值不翻译
```

### 规则

- 翻译注释（`;` 或 `#` 开头）
- 翻译配置值中的字符串
- 键名不翻译
- `[section]` 节名不翻译
- 布尔值（true/false/yes/no/on/off）不翻译
- 数字不翻译
- 保留文件结构和缩进

## 日志文件（.log）

- 翻译日志中的消息文本
- 保留时间戳、IP 地址、级别（INFO/ERROR/WARN）
- 保留协议字段、端口号
- 保留技术标识符（文件名、路径、进程名）

示例：
```
原始: 2024-01-15 14:32:05 ERROR Connection failed to database server
翻译: 2024-01-15 14:32:05 ERROR 连接到数据库服务器失败
```
