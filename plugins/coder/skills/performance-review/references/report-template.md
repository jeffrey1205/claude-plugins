## 报告输出要求

1. **终端摘要**：输出执行摘要部分，包含主要发现和优先建议
2. **完整报告**：保存为 `performance-report.md`，包含所有章节
3. **数据表格**：使用 Markdown 表格格式
4. **代码示例**：使用代码块，标注语言
5. **量化指标**：尽量提供数值，避免模糊描述
